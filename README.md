This repository contains the code to compute the exponents tau in Table 1 of the paper "Sums of distinct powers of finitely many positive integers" by G. Melfi.

By using controlling the decaying of density by both a technique described in the forward reduction Lemma and backward enhancing Lemma, the codes below give an 
estimate of tau such that the counting fonction of Sigma(Pow(A)) up to x exceeds x^{1-tau}

Here Sigma(Pow(A)) is the set of sums of distinct powers of positive integers in the multiset A.
